# CIVE-202---Project-3---Transportation-Systems

The repository contains files and information on the CIVE 202 Project #3 - Transportation Systems. This code visualizes transportation data trends in multiple types of charts.

Transportation systems are vital to the United States economy and daily life. Two companies were hired to collect extensive data to understand the transportation usage, patterns, and performance. Two data sources, National Household Travel Survey (NHTS) and Next Generation Simulation Data (NGSIM), were used to analyze traffic patterns and behaviors. Important transportation trends in the data are visualized using a bar chart, histogram, boxplot, and two different time-series plots.

To use this Python tool, open the ipynb file and follow the directions imbedded in the code. The code will allow the exportation of multiple charts used in anaysis.
